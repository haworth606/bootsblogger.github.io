Notes

Grid widget menggunakan calc() untuk mengkalkulasi lebar dan jarak widget, lihat dukungan browser untuk calc(). Menggunakan calc() adalah untuk menghindari penggunaan padding pada .widget, agar tidak merusak tampilan komponen widget lainnya, dikarenakan komponen widget lainnya seperti widget panel, widget box menerapkan CSS pada .widget, juga agar tidak mengganggu ketika Anda membuat komponen baru untuk widget.
