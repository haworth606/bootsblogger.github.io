Overview

Template dasar Bootsblogger dirancang untuk memberikan kemudahan dengan mengubahsuaikan fitur bawaan Blogger dan menambahkan fitur-fitur baru.

Blogger version

  • Menggunakan template versi 3, <html b:layoutsVersion='3'>.
  • Menggunakan gadget versi 2, <html b:defaultwidgetversion='2'> atau <b:widget version='2'>.
