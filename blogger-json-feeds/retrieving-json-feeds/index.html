Retrieving JSON feeds

Blogger mempunyai data posting, halaman statis, dan komentar dalam bentuk JSON, yang disediakan oleh Blogger Data API.

Di bawah ini adalah cara mengambil data JSON feeds dari Blogger Data API.

Notation

Description
{postId} Ganti dengan nomor unik posting.
{label} Ganti dengan nama label posting.
{type}

Ganti dengan default atau summary.

  • default: tubuh posting sesuai aslinya.
  • summary: tubuh posting yang sudah dipotong (ringkasan).
{order}

Ganti dengan published atau updated.

  • published: mengambil data JSON posting dan halaman statis berdasarkan pada waktu penerbitan.
  • updated: mengambil data JSON posting dan halaman statis berdasarkan data yang telah diperbarui.
{x} Ganti dengan nomor, yang digunakan untuk mengambil data JSON mulai dari urutan ke-x. x ≥ 1 dan tidak boleh lebih besar dari jumlah data yang ada.
{n} Ganti dengan nomor, yang digunakan untuk membatasi jumlah data JSON yang diambil, sebanyak n.
alt=json-in-script Data yang diambil berupa JSON.
{function} Ganti dengan nama-fungsi JavaScript, yang digunakan untuk mengubah data JSON menjadi markup HTML.
