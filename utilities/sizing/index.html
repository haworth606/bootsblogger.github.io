Sizing

Kelas tambahan untuk Bootstrap sizing utilities.

Responsive width

Format kelasnya tidak jauh berbeda dengan Bootstrap sizing utilities, hanya menambahkan {breakpoint}, format kelasnya adalah .w-{breakpoint}-{25, 50, 75, 100}.

Example

Ubah ukuran layar browser Anda untuk melihat perubahannya.

.w-lg-100 (100%)
.w-md-75 (75%)
.w-sm-50 (50%)
.w-25 (25%)
