Columns

Gunakan .wg-columns.

Hanya bekerja pada perangkat kecil (sm) ke atas.

  • Pada perangkat sedang (md) ke atas (≥768px) akan tampil tiga kolom.
  • Pada perangkat kecil (sm) ke atas (≥576px) akan tampil dua kolom.
  • Pada perangkat sangat kecil (xs) ke bawah (<576px) hanya tampil satu kolom.

Grid

Sistem grid untuk widget.

Grid options

Extra small
<576px 		Small
≥576px 		Medium
≥768px 		Large
≥992px 		Extra large
≥1200px
Grid behavior Horizontal at all times Collapsed to start, horizontal above breakpoints
Class prefix .wg- .wg-sm- .wg-md- .wg-lg- .wg-xl-
# of columns 12
Gutter width 30px (15px on each side of a column)
Offsets No
Column ordering No

Grid widget menggunakan calc() untuk mengkalkulasi lebar dan jarak widget. Menggunakan calc() adalah untuk menghindari penggunaan padding pada .widget, agar tidak terjadi konflik dengan CSS widget lainnya.
